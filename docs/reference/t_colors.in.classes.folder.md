# Test for colors in classes distribution for folders

Test for colors in classes distribution for folders

## Usage

``` r
t_colors.in.classes.folder(path, test = "Wilcoxon")
```

## Arguments

- path:

  path to folder

- test:

  "Wilcoxon", "wilcox" or "U" for Wilcoxon rank-sum (Mann-Whitney U),
  chisq for Chi-squared test

## Value

test results
