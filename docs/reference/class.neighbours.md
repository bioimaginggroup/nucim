# Class neighbourhood distribution

Class neighbourhood distribution

## Usage

``` r
class.neighbours(img, N, N.max = 7, cores = 1)
```

## Arguments

- img:

  Class image

- N:

  which class

- N.max:

  maximum class (default: 7)

- cores:

  number of cores used in parallel (needs parallel package)

## Value

vector of length N.max
